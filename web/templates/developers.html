API Documentation

Vostros provides a REST API for programmatic access. Authenticate using a Bearer token in the Authorization header.

Authentication

Create an API token below, then include it in your requests:

curl -H "Authorization: Bearer vst_YOUR_TOKEN" BASE_URL/api/v1/global

Endpoints

Method Path Auth Description
GET/api/v1/globalNoGlobal timeline
GET/api/v1/timelineYesHome timeline (follows)
POST/api/v1/tweetsYesCreate a tweet
GET/api/v1/tweets/{id}NoGet a single tweet
DELETE/api/v1/tweets/{id}YesDelete your tweet
GET/api/v1/users/{username}NoGet user profile
POST/api/v1/users/{username}/followYesFollow a user
DELETE/api/v1/users/{username}/followYesUnfollow a user
GET/api/v1/search?q=termNoSearch tweets

Examples

# Get global timeline
curl -H "Authorization: Bearer vst_YOUR_TOKEN" \
  -H "Accept: application/json" \
  BASE_URL/api/v1/global

# Post a tweet
curl -X POST -H "Authorization: Bearer vst_YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"content":"Hello from the API!"}' \
  BASE_URL/api/v1/tweets

# Search tweets
curl -H "Accept: application/json" \
  "BASE_URL/api/v1/search?q=hello"

Pagination

List endpoints support cursor-based pagination. Pass ?cursor=LAST_ID to get the next page. Responses include the cursor in the last item's id field.
